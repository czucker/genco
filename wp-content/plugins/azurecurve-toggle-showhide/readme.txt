== Description ==
Toggle to show/hide content (allows custom title).

[toggle]content[/toggle] to use toggle in basic format.

Set a title by using the title variable: [toggle title='Click to show/hide spoiler']content[/toggle]

== Installation ==
To install the plugin copy the <em>azurcurve-toggle-show-hide</em> folder into your plug-in directory and activate it.
